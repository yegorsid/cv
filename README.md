# Simple CV page

## Description

One-page app made with React and Chakra UI.

## Key features

- Light/Dark theme switching made with ChakraUI
- Language changing made with react-i18next

## Launch

To run locally:

``
npm run dev
``

To build locally:

``
npm run build
``
